# regua
regua 30 partes gunbound
1. Download

Baixe a versão mais recente da régua aqui:

https://github.com/Masteruler/regua/releases/download/v1/gb_rulers_combo.exe

Não é necessário instalar nada. Basta executar.

2. Como usar
2.1 Executar

• Dê dois cliques no arquivo gb_rulers_combo.exe.
• O programa abre imediatamente com a régua centralizada na tela.
• Não exige instalação, Python ou bibliotecas adicionais.

2.2 Posição inicial

Ao abrir, a régua será exibida exatamente no centro do monitor, levemente abaixo do meio, para não bloquear o campo de visão durante o jogo.

3. Ajustando o tamanho da régua

A régua foi criada para ser elástica, permitindo adaptação a qualquer resolução ou tamanho de tela.

3.1 Como ajustar

• Passe o mouse pelas bordas da régua até aparecer o ícone de redimensionar.
• Clique e arraste para aumentar ou diminuir horizontalmente.
• Ajuste até alinhar exatamente com a distância desejada no seu monitor.

3.2 Salvar o tamanho personalizado

• Ajuste o tamanho uma única vez.
• Feche o programa usando Alt + F4.
• Ao abrir novamente, a régua carregará automaticamente o último tamanho salvo.

4. Usando dentro do Gunbound
4.1 Modo Janela

Para melhor uso, coloque o Gunbound em:
• Janela
ou
• Janela sem borda

Isso permite posicionar a régua exatamente sobre o campo do jogo.

4.2 Transparência

A régua possui fundo realmente transparente, permitindo visualizar tudo no jogo enquanto utiliza os marcadores numéricos.

4.3 Movimento

A régua pode ser movida livremente pela tela:
• Clique e arraste em qualquer área neutra
• Solte quando estiver posicionada onde você deseja

5. Recomendações

• A régua foi criada como um apoio visual para que o usuário possa sobrepor um papel transparente (ou papel vegetal) sobre o monitor, traçar as marcações desejadas e utilizar a régua de forma totalmente externa, fixando o papel com uma pequena fita adesiva.
• Essa abordagem garante que cada jogador tenha sua própria régua física personalizada, sempre na proporção correta para o seu monitor.
• Ajuste a régua digital apenas para calibrar o tamanho da régua física que você irá utilizar.
• Utilize a régua digital somente como ferramenta de medição e referência.
• O tamanho ajustado é salvo automaticamente e pode ser recalibrado quando quiser.
• Mantenha o Gunbound em modo janela ou janela sem borda para facilitar o alinhamento enquanto calibra seu papel.

6. Sobre o projeto

Ferramenta desenvolvida para uso externo, sem interação com o processo do jogo.
Segura, leve, sem instalação e sem risco de detecção por sistemas anticheat, pois não injeta, lê ou modifica nada dentro do jogo.
